## 1. 개요

상품 수가 증가함에 따라 효율적인 상품 관리를 위한 내부 코드(SKU) 시스템 도입 제안

---

## 2. SKU 형식

```
CL-110-01-001
│   │   │   └── 일련번호 (관리자만 표시)
│   │   └────── 색상 코드
│   └────────── 카테고리 코드
└────────────── 브랜드 코드 (약자)
```

### 예시
- `CL-110-01-001` → 화면 표시: **CELINE 숄더백 (블랙)**
- 일련번호(001)는 관리자 권한자만 확인 가능

## 4. 구현 범위

### 4.1 관리자 페이지
- 상품 추가/수정 시 SKU 입력 필드 추가
- SKU 자동 생성 도우미 (선택형)
- 상품 목록에 SKU 표시

### 4.2 고객 화면
- 상품 상세: 브랜드 + 카테고리 + 색상만 표시
- 일련번호는 숨김 처리
