에센시아 시스템 가이드북

Premium Shopping Mall Platform Guide

이 가이드북은 에센시아 쇼핑몰 시스템을 처음 접하는 분들도 쉽게 이해하고 운영할 수 있도록 작성되었습니다.

주요 특징 (Key Features)

  • 반응형 웹 (Responsive Web): PC, 태블릿, 스마트폰 어디서든 최적화된 화면
  • 실시간 관리 (Real-time Management): 주문 및 재고 변경 사항 즉시 반영
  • 통합 관리자 (Integrated Admin): 상품, 주문, 고객, 리뷰 통합 관리

2. 관리자 매뉴얼 (Admin Manual)

쇼핑몰 운영의 핵심인 관리자 페이지 사용법입니다.
접속 주소: /admin-login

2.1 대시보드 (Dashboard)

로그인 후 가장 먼저 보이는 화면입니다. 쇼핑몰의 현재 상태를 한눈에 파악할 수 있습니다.

  • 오늘의 매출: 오늘 하루 동안의 총 판매 금액
  • 주문 현황: 결제 완료, 배송 중 등 처리해야 할 주문 건수
  • 알림: 재고 부족 상품, 신규 주문/리뷰 알림

2.2 주문 관리 (Order Management) 중요

고객의 주문을 확인하고 처리하는 가장 중요한 곳입니다.

  1. 주문 확인: 새로운 주문은 '결제완료(Paid)' 상태로 표시됩니다.
  2. 배송 처리: 상품 발송 후 '배송중(Shipped)'으로 변경하고 송장 번호를 입력합니다.
  3. 완료 처리: 배송이 끝나면 '배송완료(Delivered)'로 변경합니다.

* 참고: 주문이 '결제완료' 상태가 되면 고객에게 1%의 포인트가 자동으로 적립됩니다.

2.3 상품 및 쿠폰 관리

상품 관리

  • 상품 등록/수정 및 다중 이미지 업로드
  • 재고 5개 미만 시 '재고 부족' 알림 발생

쿠폰 관리

  • 퍼센트(%) 또는 금액(원) 할인 쿠폰 생성
  • 유효 기간 및 최소 주문 금액 설정 가능

3. 운영 규칙 (Operation Rules)

3.1 포인트 정책 (Point Policy)

  • 적립 (Accumulation): 결제 완료 시 결제 금액의 1% 자동 적립
  • 사용 (Usage): 1포인트 = 1원 (주문 시 현금처럼 사용 가능)

3.2 알림 시스템 (Notification System)

  • 재고 경고: 상품 재고 5개 미만 시 알림
  • 신규 알림: 새 주문, 리뷰, 문의 등록 시 대시보드 알림

4. 기술 가이드 (Technical Guide)

개발자 및 시스템 유지보수 담당자를 위한 정보입니다.

기술 스택 (Tech Stack)

  • Framework: Next.js 14 (App Router)
  • Language: TypeScript
  • Database: Supabase (PostgreSQL)
  • Styling: Tailwind CSS

데이터 구조 (Database Structure)

users : 회원 정보 products : 상품 정보 orders : 주문 정보 coupons : 쿠폰 정보
